# Python Exercises – Python Programming Exercises, Gently Explained

This repository contains my solutions to exercises from the book “Python Programming Exercises, Gently Explained” by Al Sweigart.

The book is available for free online:
https://inventwithpython.com/pythongently/

📚 About the Book

“Python Programming Exercises, Gently Explained” is a collection of 42 beginner-friendly programming exercises designed to help learners practice Python fundamentals through small, focused problems.

Each exercise includes:

A clear problem description
Example assert tests
Hints and conceptual explanations
Optional solution guidance

The goal is to improve programming skills through practice, not complexity or competition.

🎯 Purpose of This Repository

This repository is my personal practice log while working through the exercises.

Each solution focuses on:

Writing clean, readable Python code
Understanding core programming concepts

🧠 Topics Covered

Across the exercises, I practiced:

- Variables and data types
- Loops and conditionals
- Functions and return values
- Lists and string manipulation
- Basic algorithms (sorting, searching)
- Problem decomposition
- Edge case handling

🧪 Testing Approach

Most solutions are verified using simple assertions provided in the book, for example:

```py
assert bubble_sort([2, 1, 3]) == [1, 2, 3]
```

These tests ensure correctness without requiring external testing frameworks.
